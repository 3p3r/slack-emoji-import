# Slack Emoji Import

A slack builk emoji importer built on [puppeteer](https://pptr.dev/) heavily based on [emojipacks](https://github.com/lambtron/emojipacks).

## Usage

* `npm i -g slack-emoji-import`
* `slack-emoji-import path/to/emoji-pack.yaml`
